S T c g have different meanings

S ：(file Size differs)
M ：(Mode differs)
5 ：(MD5 sum differs)
D ：(Device major/minor number mis-match)
L ：(readLink(2) path mis-match)
U ：(User ownership differs)
G ：(Group ownership differs)
T ：(mTime differs)
c ：(config file)
d ：(documentation)
g ：(ghost file)
l ：(license file)
r ：(read me)
